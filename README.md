# 🎮 Jogo da Velha no Terminal

Jogo da velha simples em Python, jogado via terminal, com alternância entre dois jogadores.

## 🚀 Funcionalidades
- Tabuleiro interativo
- Alternância entre jogadores X e O
- Verificação de vitória ou empate

## 🧰 Tecnologias
- Python 3.x

## 📦 Como jogar
1. Execute o script:
```bash
python jogo.py
